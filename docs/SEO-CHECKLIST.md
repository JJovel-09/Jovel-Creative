## Pre-Launch SEO Tasks

Complete these **BEFORE** your website goes live.

### ✅ Technical SEO Foundations

- [ ] **Domain and Hosting**
  - Domain name purchased and configured
  - SSL certificate installed (HTTPS enabled)
  - Website loads at `https://jovelcreative.com` (not http)
  - www or non-www redirect configured (choose one, stick with it)

- [ ] **Meta Tags on Every Page**
  - [ ] Unique, descriptive title tag (50-60 characters)
  - [ ] Unique meta description (150-160 characters)
  - [ ] Meta viewport tag present
  - [ ] Canonical URL tags set correctly
  - [ ] Open Graph tags for social sharing
  - [ ] Twitter Card tags

- [ ] **Heading Structure**
  - [ ] One H1 per page (main topic)
  - [ ] H2-H6 used in proper hierarchy (no skipping levels)
  - [ ] Headers include target keywords naturally
  - [ ] Headers are descriptive and meaningful

- [ ] **Schema Markup (Structured Data)**
  - [ ] LocalBusiness schema on homepage and contact page
  - [ ] Person schema on about page
  - [ ] Service schema on services page
  - [ ] FAQPage schema on services page
  - [ ] BreadcrumbList schema on all interior pages
  - [ ] Test schema with [Google Rich Results Test](https://search.google.com/test/rich-results)

- [ ] **Image Optimization**
  - [ ] All images have descriptive ALT text
  - [ ] Image file names are descriptive (frederick-web-developer.jpg, not IMG_1234.jpg)
  - [ ] Images compressed/optimized for web
  - [ ] Width and height attributes specified
  - [ ] Lazy loading enabled for below-fold images

- [ ] **Performance Optimization**
  - [ ] CSS minified
  - [ ] JavaScript minified
  - [ ] Critical CSS inlined (if needed)
  - [ ] Defer non-critical JavaScript
  - [ ] Gzip compression enabled (.htaccess configured)
  - [ ] Browser caching enabled (.htaccess configured)
  - [ ] Page load speed under 3 seconds
  - [ ] Test with [Google PageSpeed Insights](https://pagespeed.web.dev/) - aim for 90+ score

- [ ] **Mobile Optimization**
  - [ ] Fully responsive on all screen sizes
  - [ ] Touch targets at least 44x44px
  - [ ] Text readable without zooming (16px minimum)
  - [ ] No horizontal scrolling
  - [ ] Test with [Google Mobile-Friendly Test](https://search.google.com/test/mobile-friendly)

- [ ] **Sitemap and Robots.txt**
  - [ ] `sitemap.xml` created and uploaded to root
  - [ ] Sitemap includes all important pages
  - [ ] `robots.txt` created and uploaded to root
  - [ ] robots.txt references sitemap location
  - [ ] Test sitemap access: `https://jovelcreative.com/sitemap.xml`
  - [ ] Test robots.txt access: `https://jovelcreative.com/robots.txt`

- [ ] **Internal Linking**
  - [ ] Clear navigation menu on every page
  - [ ] Footer links to all major pages
  - [ ] Contextual links in content (services → contact, portfolio → services, etc.)
  - [ ] Breadcrumbs on interior pages
  - [ ] No broken internal links

- [ ] **Accessibility (Helps SEO)**
  - [ ] Skip to main content link
  - [ ] All images have ALT text
  - [ ] Forms have proper labels
  - [ ] ARIA labels where needed
  - [ ] Sufficient color contrast
  - [ ] Keyboard navigation works

### ✅ Content SEO

- [ ] **Homepage**
  - [ ] H1 includes "Frederick" and primary keyword
  - [ ] Mentions service area (Frederick & Montgomery County) multiple times
  - [ ] Clear value proposition above the fold
  - [ ] Strong calls-to-action
  - [ ] Contact information visible in footer

- [ ] **Services Page**
  - [ ] Each service package clearly described
  - [ ] Includes pricing (transparency helps conversions)
  - [ ] FAQs with common questions
  - [ ] Local keywords naturally integrated
  - [ ] Clear CTAs to contact page

- [ ] **Portfolio Page**
  - [ ] Project titles are descriptive
  - [ ] Images have descriptive ALT text
  - [ ] Each project describes benefits/results
  - [ ] Technologies/approach explained

- [ ] **About Page**
  - [ ] Personal story establishes credibility
  - [ ] Mentions local experience (Montgomery County Government)
  - [ ] Explains "why custom code vs WordPress"
  - [ ] Clear process description
  - [ ] Service area mentioned

- [ ] **Contact Page**
  - [ ] NAP (Name, Address, Phone) clearly displayed
  - [ ] Contact form with clear labels
  - [ ] Business hours listed
  - [ ] Google Map embedded

### ✅ Final Pre-Launch Checks

- [ ] All pages spell-checked
- [ ] All links tested (use [Broken Link Checker](https://www.brokenlinkcheck.com/))
- [ ] Test on multiple browsers (Chrome, Firefox, Safari, Edge)

## Monthly SEO Maintenance

Do these tasks **EVERY MONTH** to maintain and improve rankings.

### First Monday of Each Month

- [ ] **Google Search Console Review**
  - [ ] Check for crawl errors - fix any issues
  - [ ] Review search queries - what keywords are bringing traffic?
  - [ ] Check average position for target keywords - improving?
  - [ ] Look for coverage issues - any pages not indexed?
  - [ ] Review mobile usability - any mobile issues?

- [ ] **Google Analytics Review**
  - [ ] Total visitors - increasing month over month?
  - [ ] Top traffic sources - is organic search growing?
  - [ ] Most viewed pages - which content resonates?
  - [ ] Bounce rate - is it improving?
  - [ ] Conversions - form submissions, phone clicks

- [ ] **Ranking Check**
  - [ ] Google your target keywords from Frederick, MD area
  - [ ] Note your position for each keyword
  - [ ] Track changes month over month

- [ ] **Backlink Check** (Every 3 Months)
  - [ ] Use free tool like Bing Webmaster Tools
  - [ ] Check for new backlinks
  - [ ] Disavow any spammy links

- [ ] **Content Update**
  - [ ] Update at least one page with fresh content
  - [ ] Add a new portfolio item (if available)
  - [ ] Update pricing if changed
  - [ ] Check all dates/copyright years are current

- [ ] **Competitor Check** (Every 3 Months)
  - [ ] Google your target keywords
  - [ ] Check top 3 competitors' sites
  - [ ] What are they doing well?
  - [ ] What topics are they covering that you're not?
  - [ ] Find content gaps and opportunities

## Content Optimization Guidelines

### Keyword Placement Best Practices

For each target keyword on a page:

- [ ] **Title Tag** - Include primary keyword near the beginning
- [ ] **H1** - Include primary keyword naturally
- [ ] **First Paragraph** - Mention primary keyword in first 100 words
- [ ] **Throughout Content** - Use variations naturally (2-3% keyword density)
- [ ] **H2/H3 Subheadings** - Include related keywords in some headings
- [ ] **Image ALT Text** - Include keywords where relevant
- [ ] **URL** - Use keyword in page URL if possible (e.g., `/services-web-developer-frederick`)
- [ ] **Meta Description** - Include primary keyword

### Content Quality Checklist

- [ ] **Unique** - Not copied from anywhere
- [ ] **Valuable** - Answers user questions or solves problems
- [ ] **Well-Written** - Proper grammar, spelling, punctuation
- [ ] **Scannable** - Short paragraphs, bullet points, headings
- [ ] **Actionable** - Clear next steps for the reader
- [ ] **Comprehensive** - Covers topic thoroughly (500+ words for key pages)
- [ ] **Updated** - Current information (no outdated dates/facts)

## Warning Signs: When to Investigate

- ⚠️ **Traffic suddenly drops** - Check Search Console for penalties or indexing issues
- ⚠️ **Rankings suddenly drop** - Check if a competitor launched new content
- ⚠️ **High bounce rate** - Page content may not match what users expect
- ⚠️ **Low conversion rate** - CTAs may need to be stronger or more visible
- ⚠️ **Pages not indexing** - Technical issues with robots.txt or site structure
